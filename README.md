# jenkins-final-task




## features
Web app that parse Ynet breaking news section Xml file 
## how to run
1.install java 17.0.3 version:
sudo apt install openjdk-17-jre

2.build the app:
mvn clean package

3. run the app:
java -jar target/jenkins-closing-task-0.0.1-SNAPSHOT.jar
